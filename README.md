## 📘 About the Project  

This repository contains a complete GitHub learning guide designed for Information Science and Engineering (ISE) students (2026).

It covers Git and GitHub concepts from beginner to advanced level, including real-world workflows, collaboration techniques, deployment using GitHub Pages, and interview preparation.

---

## 🎯 Key Features  

- Beginner to Advanced Git & GitHub concepts  
- 60+ Git commands with examples  
- Branching, merging & conflict resolution  
- Collaboration using Pull Requests  
- GitHub Pages deployment  
- 3 hands-on projects  
- Interview preparation & viva questions  
- Cheat sheet and professional tips
